## Задание
Оформление баг-репортов на найденные баги на сайте, тестирование функции добавления отзыва к товару.

#### Ознакомиться с баг-репортами можно во [вкладке Issues](https://github.com/diananaum/Bug-Reports/issues) или с каждым отдельно:
1. [Цифра "7" вставляется в поле ввода в поле "Телефон" при клике на плейсхолдер кода страны](https://github.com/diananaum/Bug-Reports/issues/1)
2. [Переход в меню личного кабинета при нажатии на кнопку "Регистрация карты"](https://github.com/diananaum/Bug-Reports/issues/2)
3. [Некорректная сортировка результатов поиска в поле "Поиск города" при вводе значения "Моск"](https://github.com/diananaum/Bug-Reports/issues/3)
4. [Некорректное отображение города на карте при просмотре адресов салонов во вкладке "На карте" и изменении города при резервировании товара в салоне](https://github.com/diananaum/Bug-Reports/issues/4)
5. [Бесконечное увеличение размера поля ввода "Ваш отзыв"](https://github.com/diananaum/Bug-Reports/issues/5)
6. [Поле "Ваша оценка" не выделяется как обязательное для заполнения](https://github.com/diananaum/Bug-Reports/issues/6)
7. [Сообщение о публикации отзыва при отправке отзыва с не отмеченным чекбоксом "Я соглашаюсь на публикацию моего отзыва на сайте henderson.ru"](https://github.com/diananaum/Bug-Reports/issues/7)
8. [Отправка отзыва с не отмеченном чекбоксе "Я соглашаюсь на обработку моих персональных данных и ознакомлен(а) с политикой конфиденциальности"](https://github.com/diananaum/Bug-Reports/issues/8)
